This project implements Logistic Regression, a supervised machine learning algorithm used for binary classification problems such as Yes/No or Pass/Fail predictions.

Concepts Covered:
Data preprocessing
Missing value handling
Outlier handling using IQR
Feature encoding and scaling
Sigmoid and Logit functions
Model training and prediction
Gradient Descent
Ridge & Lasso Regularization
Evaluation metrics:
Accuracy
Precision
Recall
F1-Score
Confusion Matrix
ROC Curve
Technologies Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Streamlit
Conclusion:
Logistic Regression is an effective classification algorithm that predicts probabilities and helps solve real-world classification problems efficiently.
